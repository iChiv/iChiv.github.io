## Overview

Repair Below From Above is a VR game where players have to complete puzzles in a closed box based on information from the monitor above.

The game was created for **Epic Games MegaJam 2022** with the theme **"As Above So Below"**.

I was responsible for **concept design, programming, sound, animation, visual effects, testing, and optimization** for this project. This was also my first time working on a VR game in Unreal Engine 5.
