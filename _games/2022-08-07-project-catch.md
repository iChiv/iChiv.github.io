## Overview

Project Catch is a collaborative game project featuring unique photo-taking and projection mechanics.

![Project Catch Screenshot](/assets/images/projects/project-catch-2.png)

I was responsible for **game mechanics and level design**, **programming** and **project resource management**.

## Why I Made This Game

The idea for the game came from a brainstorming session. Initially we had different ideas about the game, but our goal was the same - to make a game with fun and substance.

We narrowed down the keywords to 5 to ensure the game was viable:

- **Cartoon rendering** – Project Catch uses this as the basis for building scenes
- **Dimensional enhancement** – the logic of interaction between taking pictures and projection
- **Scene Mechanics** – the components that make up the levels
- **Outer Wilds** – we want Project Catch to have a dynamic and exploratory world
- **Human Motion Interaction** – players can experience the game via VR/AR or Eye-tracking devices
